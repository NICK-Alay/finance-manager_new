1 через Docker Compose 
   - открыть терминал в папке проекта C:\"весь путь"\Спортивное программирование
   - выполнить команду: docker compose up --build
   - после запуска сервис будет доступен по адресу: http://localhost:8000/docs

2 Запуск напрямую через Python
   - откройте терминал в папке проекта
   - активировать виртуальное окружение, если нужно
   - выполните команду "c:/"весь путь"/Спортивное программирование/.venv/Scripts/python.exe" "c:/"весь путь"/Спортивное программирование/zadanie.py" или через uvicorn: "c:/"весь путь"/Спортивное программирование/.venv/Scripts/python.exe" -m uvicorn zadanie:app --reload

3 основные API "маршруты" 
   - POST /categories - создать категорию
   - GET /categories - получить список категорий
   - PUT /categories/{id} - изменить категорию
   - DELETE /categories/{id} - удалить категорию
   - POST /operations - создать операцию доход или расход
   - GET /operations - получить список операций с фильтрацией и сортировкой
   - GET /operations/{id} - получить операцию по id
   - PUT /operations/{id} - обновить операцию
   - DELETE /operations/{id} - удалить операцию
   - GET /balance - получить суммарный доход, расход и остаток 

4 Параметры запроса для GET /operations   
   - date_from - дата начала периода 			"например 2026-05-01"
   - date_to - дата конца периода
   - category_id- шid категории
   - type - income или expense
   - sort_by - date, amount, type или category
   - sort_order - asc или desc


При первом запуске создаются категории как по умолчанию или основа - Еда, Транспорт, Развлечения 
Баланс автоматически считается как сумма доходов минус сумма расходов
Ввся база данных хранится в файле database.db
